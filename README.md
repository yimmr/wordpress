## 介绍

impack/wordpress（简称 imwp）是用面向对象方式开发 WordPress 主题的小型类库。它把零散的函数组织成面向不同功能的类，使用这些类可以快速实现主题功能，同时也让代码变得简洁，更容易维护，有效提升开发者的编码体验。

## 安装方式

该包仅适用主题，在主题目录下使用 composer 引入包，若未初始化包管理，先执行 `composer init` 再引入包：

```bash
composer require impack/wordpress
```
